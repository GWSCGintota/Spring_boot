# 6. System Requirements and Tools

For Spring Boot 4.1:

- Java 17 or later is required
- Maven 3.6.3 or later is supported
- Supported Gradle 8 and 9 versions can be used
- Java 21 LTS is a practical choice for new projects

Recommended tools:

- JDK 21
- IntelliJ IDEA, Eclipse, VS Code, or Spring Tools
- Maven or Gradle
- Git
- Postman, Insomnia, or `curl`
- Docker
- PostgreSQL, MySQL, or another supported database

Check Java:

```bash
java -version
```

Check Maven:

```bash
mvn -version
```

Check Gradle:

```bash
gradle -version
```
