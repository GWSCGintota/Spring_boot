# 41. Official References
---

## Official Spring Resources

### 1. Spring Boot Project

The main Spring Boot project page provides an overview of Spring Boot, its purpose, current releases, and links to documentation.

- [Spring Boot Project](https://spring.io/projects/spring-boot)

### 2. Spring Boot Documentation

The official reference documentation explains Spring Boot features, configuration, auto-configuration, dependency management, testing, deployment, and production capabilities.

- [Spring Boot Documentation](https://docs.spring.io/spring-boot/)

### 3. Spring Boot System Requirements

This page provides the supported Java, Maven, Gradle, Servlet container, and GraalVM versions for the selected Spring Boot release.

- [Spring Boot System Requirements](https://docs.spring.io/spring-boot/system-requirements.html)

### 4. Spring Initializr

Spring Initializr generates a new Spring Boot project with the selected language, build tool, Java version, packaging type, and dependencies.

- [Spring Initializr](https://start.spring.io/)

### 5. Spring Framework Documentation

The Spring Framework reference covers the core container, dependency injection, Spring MVC, WebFlux, data access, transactions, validation, testing, and integration features.

- [Spring Framework Documentation](https://docs.spring.io/spring-framework/reference/)

### 6. Spring Data JPA

Spring Data JPA simplifies database access by providing repository interfaces, derived queries, pagination, sorting, projections, and custom query support.

- [Spring Data JPA](https://spring.io/projects/spring-data-jpa)

### 7. Spring Security

Spring Security provides authentication, authorization, password encoding, method security, OAuth 2.0, OpenID Connect, and resource-server support.

- [Spring Security](https://spring.io/projects/spring-security)

### 8. Spring Guides

The Spring Guides contain practical tutorials for building REST services, accessing databases, securing applications, creating batch jobs, using messaging, and deploying Spring applications.

- [Spring Guides](https://spring.io/guides)

---

## Recommended Learning Order

1. Generate a project using Spring Initializr.
2. Read the Spring Boot reference documentation.
3. Study dependency injection and bean management in the Spring Framework documentation.
4. Learn REST API development using Spring MVC.
5. Study persistence using Spring Data JPA.
6. Add authentication and authorization using Spring Security.
7. Follow the official Spring Guides for practical examples.
8. Review system requirements before upgrading Spring Boot or Java.
